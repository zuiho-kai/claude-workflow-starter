# 仓库经验入口

## 什么时候查这里

- 用户给了 upstream 名、URL、显示名或本地目录名，需要把它映射到 canonical `repos/<slug>/`。
- 通用方法不足以描述仓库自己的代码、命令、规则或模型。

## 不放什么

- 换到其他仓库仍然成立的通用方法；这些放 `framework/`。
- canonical slug 已在本轮验证时，不要为了导航再读本页。
