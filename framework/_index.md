# 通用经验入口

## 什么时候查这里

- 需要的方法换到其他仓库后仍然成立。
- 还没有确定具体仓库，先查通用做法。

## 不放什么

- 某个仓库、代码模块或模型独有的命令和结论。
- 当前机器的地址、路径、账号和凭据。
