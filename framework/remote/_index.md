# 远端任务

## 什么时候查这里

- 使用 SSH、容器、tmux、超时、清理或远端长跑任务。

## 不放什么

- 真实服务器地址、账号、私钥和某台机器的 cache 路径。
- 某个仓库专有的测试命令。
